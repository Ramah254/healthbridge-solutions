# HealthBridge Solutions Website

A complete, professional multi-page website for HealthBridge Solutions - a healthcare automation company based in Kenya.

## Overview

This website is built with pure HTML, CSS, and JavaScript (no frameworks) and includes:

- **6 Pages**: Home, About, Services, Contact, Privacy Policy, Terms of Service
- **Responsive Design**: Mobile-first approach that works on all devices
- **Modern UI**: Clean, professional design with blue (#1A56DB) and orange (#F97316) color scheme
- **SEO Optimized**: Meta tags, Open Graph tags, and structured data for search engines
- **Fast Loading**: No unnecessary libraries or dependencies

## How to Use

### Local Development
1. Download or clone the repository
2. Open any HTML file in your web browser
3. No build process or server required

### Deployment
1. Upload all files to your web server
2. Ensure all files are in the same directory
3. The main entry point is `index.html`

### Customization

#### Changing Colors
Edit the CSS variables in `styles.css`:
```css
:root {
  --primary-blue: #1A56DB;
  --primary-orange: #F97316;
  /* ... other variables */
}
```

#### Updating Content
- Edit the HTML files directly to update text content
- Replace placeholder images with actual images
- Update contact information in all files

#### Adding Pages
1. Create a new HTML file
2. Copy the navigation and footer from existing pages
3. Link to the new page in the navigation menu
4. Add styles to `styles.css` if needed
